Wasp Balancer

This project was initially created as a research project on applying a bio-inpsired algorithm on a real world problem. I used a polish wasp algorithm as the routing logic for a load balancer. The idea behind a polish wasp algorithm is that each unit starts off generic and specializes in a task the more it does it. Since web servers will cache previously requested items, then sending the same type of requests the same machine will see better performance. To see the code and the results checkout the Github Repo
